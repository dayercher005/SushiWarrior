## How to Run
1. Clone the repository:
```bash
git clone https://github.com/dayercher005/SushiWarrior
```
2. Install dependencies:
```bash
npm install
```
3. Start development server:
```bash
npm run start
```
4. Open `http://localhost:8080` in your browser
